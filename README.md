
## reant-cli

to create a basic web project with react, react-router, redux, ant design for PC

## 使用教程

### 安装
```
npm install -g reant-cli
```

### 创建项目并安装依赖
```
reant create demo
cd demo
npm install
```

### 运行开发环境
```
npm start
```

浏览器打开： [http://localhost](http://localhost) 即可

### 构建
```
npm run build
```
